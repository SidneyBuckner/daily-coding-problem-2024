Daily Coding Problem #3: Convert Array to BST

Given an array of integers, create a binary search tree.

For example:

  1. Input: [10, 4, 8, 11, 15, 30, 1, 3, 5] - Output: 10
  2. Input: [] - Output: null
  3. Input: [5] - Output: 5
  4. Input: ["apple", "banana", "cherry", "date", "elderberry"] - Output: Apple (lexicographically first)
