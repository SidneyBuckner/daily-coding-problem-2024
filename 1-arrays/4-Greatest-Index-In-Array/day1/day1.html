Daily Coding Problem #4: Greatest Index Value in an array

Given an integer array nums, locate the largest value in the array and return its index.

For example:

  1. input: [-2,1,-3,4,-1,2,1,-5,4] - output: 8
  2. input: [34, 105, -50, 42, 14, -5, 86] - output: 1
  3. input: [-5, -1, 8, -9] - output: 2
