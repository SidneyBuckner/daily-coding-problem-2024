Daily Coding Problem #2: Locate the smallest window to be sorted

Given an integer array nums, find the contiguous subarray (containing at least one number) which has the largest sum and return its sum.

For example:

  1. input: [-2,1,-3,4,-1,2,1,-5,4] - output: 6
  2. input: [34, -50, 42, 14, -5, 86] - output: 137
  3. input: [-5, -1, -8, -9] - output: 0
